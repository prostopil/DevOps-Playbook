Базовые команды Git

1. Первоначальная настройка нового проекта

cd путь/к/вашей/папке

Переход в папку проекта

git init

Инициализация Git репозитория

2. Добавление и отправка файлов

git add .

Добавить все файлы

git commit -m "описание изменений"

Создать коммит

git push

Отправить изменения на сервер

Подробная инструкция по работе с Git

1. Начало работы с новым проектом

  1. cd C:\Projects\MyProject

    Перейдите в папку проекта (Windows)

    cd ~/Projects/MyProject

    Перейдите в папку проекта (Linux/Mac)

  2. git init

    Инициализируйте Git репозиторий

  3. git remote add origin https://github.com/username/repository.git

    Привяжите удаленный репозиторий (замените URL на ваш)

2. Добавление изменений в существующий проект

  1. git status

    Проверьте статус изменений

  2. git add имя_файла

    Добавьте конкретный файл

    git add .

    Или добавьте все измененные файлы

  3. git commit -m "подробное описание изменений"

    Создайте коммит с описанием изменений

  4. git pull

    Получите последние изменения с сервера (важно сделать перед push)

  5. git push

    Отправьте изменения на сервер

Расширенные команды Git

1. Работа с ветками

git branch new-feature

Создает новую ветку

git checkout -b feature-branch

Создает и переключается на новую ветку

2. Слияние и разрешение конфликтов

git merge feature-branch

Сливает указанную ветку в текущую

git mergetool

Запускает инструмент разрешения конфликтов

3. История и логи

git log --graph --oneline --all

Показывает графическое представление истории коммитов

git reflog

Показывает историю всех действий в репозитории

4. Отмена изменений

git reset --hard HEAD~1

Отменяет последний коммит

git revert HEAD

Создает новый коммит, отменяющий изменения последнего коммита
